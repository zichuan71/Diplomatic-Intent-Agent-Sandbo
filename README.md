# Diplomatic-Intent-Agent-Sandbo
基于大模型与云边协同架构的外交话语意图对抗沙盘
本项目是面向复杂国际地缘政治场景的外交话语智能分析与对抗演练系统，核心解决：外媒记者提问中的高敏感意图精准识别;多智能体模拟的外交话语攻防演练;云边协同架构下的低延迟、高并发意图推理;
系统采用 边缘侧小模型实时感知 + 云端大模型对抗生成 混合架构，已部署于 K3S 多云混合集群，可用于学术研究、政策推演、舆情预警等场景。
1. 边缘侧：本地意图感知基座（已完成）
基座模型：hfl/chinese-roberta-wwm-ext-large
训练范式：Data-Centric AI + 人工修正标签 + 5-Fold 交叉验证
优化策略：动态类别权重解决长尾分布、类别不平衡问题
性能指标：Macro-F1 优异，具备工业级部署能力
部署环境：RTX 单机推理节点 / K3S 边缘集群
2. 云侧：多智能体对抗演练沙盘（开发中）
引擎：轻量级自研多智能体调度引擎（剥离商业化冗余组件）
核心逻辑：
虚拟外媒记者组（多角色并发生成）
中方发言人智能应答
思维链反思（CoT Reflection）
自动生成隐蔽攻击意图样本（归责转移、预设前提、话术陷阱等）
对边缘模型进行极限边界压力测试
特性：异步并发、高吞吐、自动对抗数据集生成
